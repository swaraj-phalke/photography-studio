# LensCraft - Cinematic Photography Portfolio

A premium, award-winning photography portfolio website built with React, Framer Motion, and Tailwind CSS. Features cinematic animations, interactive galleries, and a luxury user experience.

## ✨ Features

### Design & Aesthetics
- **Cinematic Design**: Black, white, and gold color scheme with luxury aesthetics
- **Premium Typography**: Playfair Display for headings, Lato for body text
- **High-Quality Imagery**: Full-screen hero images with parallax scrolling

### Interactive Components
- **Hero Section**: Cinematic slideshow with animated taglines
- **Portfolio Gallery**: Filterable gallery with hover effects and lightbox popup
- **About Section**: Timeline of achievements with fade-in animations
- **Testimonials**: Carousel with smooth transitions and client reviews
- **Contact Form**: Multi-step animated booking form with micro-interactions

### Animations & Effects
- **Framer Motion**: Smooth page transitions and scroll-triggered animations
- **Hover Effects**: Tilt, zoom, and fade effects on interactive elements
- **Parallax Scrolling**: Immersive scrolling experience
- **Cinematic Overlays**: Premium gallery-like feel

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```

4. **Preview production build**
   ```bash
   npm run preview
   ```

## 🛠️ Tech Stack

- **React 19** - Modern React with latest features
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Production-ready motion library
- **Lucide React** - Beautiful & consistent icon toolkit
- **React Intersection Observer** - Scroll-triggered animations

## 🎨 Customization

### Colors
The color scheme can be customized in `src/index.css`:
- Primary: Black (#000000)
- Secondary: White (#FFFFFF)
- Accent: Amber/Gold (#F59E0B)

### Content
Update the following files to customize content:
- `src/components/HeroSection.jsx` - Hero images and taglines
- `src/components/Portfolio.jsx` - Portfolio items and categories
- `src/components/About.jsx` - About content and timeline
- `src/components/Testimonials.jsx` - Client testimonials
- `src/components/Contact.jsx` - Contact information

### Images
Replace placeholder images with your own:
- Use high-quality images (1920x1080 or higher)
- Optimize images for web (WebP format recommended)
- Update image URLs in component files

## 🚀 Deployment

### Vercel (Recommended)
1. Push code to GitHub
2. Connect repository to Vercel
3. Deploy automatically

### Netlify

The built files in the `dist` folder can be deployed to any static hosting service.
